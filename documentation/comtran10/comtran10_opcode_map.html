COMTRAN TEN

Opcode Decode Map

How to read: find the upper hex digit of the opcode in the MSD column (left), then find the lower hex digit in the LSD row (top). The cell shows the instruction.
Rows 0-1: non-memory instructions -- all 8 bits identify the instruction. Cells marked INVALID are explicitly undefined.
Rows 2-F: memory instructions -- upper 5 bits identify the instruction, lower 3 bits are address modifiers. The sub-header row shows: p0=page 0, p1=page 1, p2=page 2, p3=page 3, ·x=indexed.
