Miscellaneous

NOP — No Operation

none
ModeSyntaxOpcodeBytesCycles
inherentNOP$1212
Condition Codes
HNZVC
—————

Does nothing for 2 cycles. Used for timing loops, code patching placeholders, and alignment.

ABX — Add Accumulator B to X

X ← X + B (unsigned)
ModeSyntaxOpcodeBytesCycles
inherentABX$3A13
Condition Codes
HNZVC
—————

B is treated as unsigned (0-255). No flags affected. Equivalent to LEAX B,X but faster (3 vs 4+ cycles) and does not affect Z. Useful for indexing into a table using B as an unsigned offset.

SEX — Sign Extend B into A

A ← sign extend of B (A ← $FF if B7=1, else A ← $00)
ModeSyntaxOpcodeBytesCycles
inherentSEX$1D12
Condition Codes
HNZVC
—↕↕——

Sign extends the 8-bit value in B into the 16-bit D register. If B is negative (bit 7 set), A is set to $FF; otherwise A is cleared. Allows treating B as a signed 16-bit value in D.

CWAI — AND Condition Codes and Wait for Interrupt

CC ← CC AND M; push all registers; wait for interrupt
ModeSyntaxOpcodeBytesCycles
immediateCWAI #n$3C220+
Condition Codes
HNZVC
↕↕↕↕↕

Clears selected CC bits via AND mask, then pushes all registers (sets E flag in CC), then halts execution until an unmasked interrupt arrives. Faster interrupt response than SYNC because all registers are pre-saved. Minimum 20 cycles before interrupt vector is fetched.

SYNC — Synchronize with Interrupt

halt until interrupt
ModeSyntaxOpcodeBytesCycles
inherentSYNC$1312+
Condition Codes
HNZVC
—————

Halts execution and waits for any interrupt. If the interrupt is masked, execution resumes at the next instruction after 2 cycles (without servicing the interrupt). If unmasked, the interrupt is serviced normally. Used for precise timing synchronization.

SWI — Software Interrupt 1

push all registers; PC ← M[$FFFA:$FFFB]
ModeSyntaxOpcodeBytesCycles
inherentSWI$3F119
Condition Codes
HNZVC
—————

Pushes all registers with E set, then vectors through $FFFA/$FFFB. Under OS-9 this is the syscall mechanism — the byte following SWI is the syscall number.
