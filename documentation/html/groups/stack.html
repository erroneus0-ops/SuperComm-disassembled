PSHS — Push Registers to S Stack

S ← S - n; M[S] ← registers
ModeSyntaxOpcodeBytesCycles
inherentPSHS reglist$3425+n
Condition Codes
HNZVC
—————

Postbyte is a bitmask of registers to push. Registers are pushed in a fixed order regardless of bit order in postbyte: PC, U, Y, X, DP, B, A, CC (highest address to lowest). Cycles = 5 + 1 per 8-bit register + 2 per 16-bit register. Cannot push S itself.

PULS — Pull Registers from S Stack

registers ← M[S]; S ← S + n
ModeSyntaxOpcodeBytesCycles
inherentPULS reglist$3525+n
Condition Codes
HNZVC
↕↕↕↕↕

Pulls registers in reverse push order: CC, A, B, DP, X, Y, U, PC. PULS PC is equivalent to RTS. If CC is in the list, all condition codes are restored from the stack. Cycles = 5 + 1 per 8-bit register + 2 per 16-bit register.

PSHU — Push Registers to U Stack

U ← U - n; M[U] ← registers
ModeSyntaxOpcodeBytesCycles
inherentPSHU reglist$3625+n
Condition Codes
HNZVC
—————

Same as PSHS but uses U stack. Postbyte bit 6 refers to S (not U). Cannot push U itself.

PULU — Pull Registers from U Stack

registers ← M[U]; U ← U + n
ModeSyntaxOpcodeBytesCycles
inherentPULU reglist$3725+n
Condition Codes
HNZVC
↕↕↕↕↕

Same as PULS but uses U stack. PULU PC jumps to the address on top of the U stack.
