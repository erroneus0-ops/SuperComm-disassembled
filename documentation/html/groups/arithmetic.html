MUL — Unsigned Multiply

D ← A × B (unsigned)
ModeSyntaxOpcodeBytesCycles
inherentMUL$3D111
Condition Codes
HNZVC
——↕—↕

Multiplies unsigned 8-bit A by unsigned 8-bit B. 16-bit result in D. C is set to bit 7 of B (the MSB of the result's low byte) — useful for rounding. Z reflects whether result is zero.

DAA — Decimal Adjust Accumulator A

A ← BCD adjust of A
ModeSyntaxOpcodeBytesCycles
inherentDAA$1912
Condition Codes
HNZVC
—↕↕?↕

Adjusts A after BCD addition (ADDA or ADCA) to produce a valid BCD result. Uses H and C flags set by the preceding addition. V is undefined after DAA. Must only be used after addition — not subtraction.
