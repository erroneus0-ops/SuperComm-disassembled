Load Effective Address

LEAX — Load Effective Address into X

X ← EA
ModeSyntaxOpcodeBytesCycles
indexedLEAX expr,R$302+4+
Condition Codes
HNZVC
——↕——

Computes the effective address and loads it into X — no memory access. Z is set if the computed address is zero. Used for PC-relative data addressing (LEAX label,PCR), pointer arithmetic, and fast 16-bit addition (LEAX n,X adds n to X in place).

LEAY — Load Effective Address into Y

Y ← EA
ModeSyntaxOpcodeBytesCycles
indexedLEAY expr,R$312+4+
Condition Codes
HNZVC
——↕——

Same as LEAX but loads Y. Z reflects whether result is zero.

LEAS — Load Effective Address into S

S ← EA
ModeSyntaxOpcodeBytesCycles
indexedLEAS expr,R$322+4+
Condition Codes
HNZVC
—————

Does not affect Z. Used to adjust the stack pointer — LEAS -n,S reserves n bytes on the stack; LEAS n,S releases them.
