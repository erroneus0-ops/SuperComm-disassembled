Increment and Decrement

INCA — Increment Accumulator A

A ← A + 1
ModeSyntaxOpcodeBytesCycles
inherentINCA$4C12
Condition Codes
HNZVC
—↕↕↕—

C is not affected — use ADDA #1 if carry is needed. V is set if A was $7F (overflow from +127 to -128 in signed arithmetic).

DECA — Decrement Accumulator A

A ← A - 1
ModeSyntaxOpcodeBytesCycles
inherentDECA$4A12
Condition Codes
HNZVC
—↕↕↕—

C is not affected. V is set if A was $80 (overflow from -128 to +127). Classic loop counter idiom: load count, loop body, DECA, BNE loop.
