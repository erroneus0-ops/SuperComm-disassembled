Postbyte encodes source (high nibble) and destination (low nibble). Transferring between different-sized registers: 8-bit to 16-bit zero-extends; 16-bit to 8-bit takes low byte. CC and DP are affected only when transferred to. When CC is the destination, all flags are set from the transferred value.
