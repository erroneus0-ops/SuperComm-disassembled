Transfer and Exchange

TFR — Transfer Register to Register

R1 ← R2
ModeSyntaxOpcodeBytesCycles
inherentTFR R1,R2$1F26
Condition Codes
HNZVC
↕↕↕↕↕

Postbyte encodes source (high nibble) and destination (low nibble). Transferring between different-sized registers: 8-bit to 16-bit zero-extends; 16-bit to 8-bit takes low byte. CC and DP are affected only when transferred to. When CC is the destination, all flags are set from the transferred value.

EXG — Exchange Registers

R1 ↔ R2
ModeSyntaxOpcodeBytesCycles
inherentEXG R1,R2$1E28
Condition Codes
HNZVC
↕↕↕↕↕

Same postbyte encoding as TFR. Both registers must be the same size. Exchanging different-sized registers produces undefined results on 6809; 6309 handles it differently.
