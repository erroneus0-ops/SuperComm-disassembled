LSLA — Logical Shift Left Accumulator A

C ← b7 ← b6...b0 ← 0
ModeSyntaxOpcodeBytesCycles
inherentLSLA$4812
Condition Codes
HNZVC
—↕↕↕↕

Also called ASLA. Shifts all bits left one position; bit 7 goes to C, bit 0 filled with 0. Equivalent to multiplying by 2. V is set if the sign changes (bit 7 XOR bit 6 before shift).

LSRA — Logical Shift Right Accumulator A

0 → b7...b0 → C
ModeSyntaxOpcodeBytesCycles
inherentLSRA$4412
Condition Codes
HNZVC
—0↕—↕

Shifts all bits right one position; bit 0 goes to C, bit 7 filled with 0. Equivalent to dividing unsigned value by 2. N is always cleared (bit 7 is always 0 after). Unlike ASRA, does not propagate sign.

ASRA — Arithmetic Shift Right Accumulator A

b7 → b7...b0 → C (sign bit propagates)
ModeSyntaxOpcodeBytesCycles
inherentASRA$4712
Condition Codes
HNZVC
—↕↕—↕

Bit 7 is reproduced (sign extension). Bit 0 goes to C. Equivalent to dividing signed value by 2, rounding toward negative infinity. V is not affected.

ROLA — Rotate Left Accumulator A through Carry

C ← b7 ← b6...b0 ← C
ModeSyntaxOpcodeBytesCycles
inherentROLA$4912
Condition Codes
HNZVC
—↕↕↕↕

9-bit rotation through carry. Old C goes to bit 0; bit 7 goes to new C. Used for multi-byte shifts — ROL/ROR pairs can shift values wider than 8 bits.
