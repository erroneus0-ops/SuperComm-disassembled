HNZVC
—————

Never branches. Equivalent to a 2-byte NOP. Useful for patching — replace BEQ/BNE with BRN to disable a branch without changing code size.

RTI — Return from Interrupt

Pull CC,A,B,DP,X,Y,U,PC from stack
ModeSyntaxOpcodeBytesCycles
inherentRTI$3B16/15
Condition Codes
HNZVC
↕↕↕↕↕

Restores CPU state from S stack. If the E (Entire) flag in the stacked CC is set, all 9 registers are restored (15 cycles). If E is clear, only CC and PC are restored (6 cycles — fast interrupt return). See CWAI and SYNC for interrupt handling.
